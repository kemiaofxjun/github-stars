# Ceta Storical House | 🏠汐塔故事屋

## Preview | 预览

框架：[VitePress](https://vitepress.dev/)

主题：[Curve](https://github.com/imsyy/vitepress-theme-curve)

## Usage | 用法

```bash
git clone https://github.com/SinzMise/blog.git
yarn # npm install
yarn build
```

### Publish | 日常发布

推送至 GitHub，使用 Actions 生成静态页面并推送 main 分支
