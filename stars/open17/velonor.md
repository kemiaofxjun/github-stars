## ⚡ 一键启动

自动安装+配置主题,一行即可!

```bash
# my-blog换为你的项目名
npm create @velonor/blog my-blog
```

现在你的博客已经创建完成啦, 你可以直接在 VitePress 中使用它:

```bash
npm i
npm run dev
```

更多配置与使用指南请参考[文档](https://vitepress.open17.vip/guide/0-intro/)

## ✨ 为什么选择 Velonor？

Velonor 是一套专为 VitePress 设计的现代化博客系统，在保持vitepress强大功能的基础上，为你带来全新的博客体验。


## 🌟 丰富功能一览

Velonor 提供了一个默认的博客主题, 你可以直接在 VitePress 中使用它, 提供包括但不限于以下功能：

- 博客主页
- 标签分类
- 归档统计
- 全文搜索
- RSS支持
- 评论系统
- 主题自定义
- 文档页面
- ...
